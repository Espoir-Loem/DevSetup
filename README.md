# PROJET 1.1:

> Ce projet est le Premier Environnement Dev chez Akieni Academy 

## pourtant sur:

Installer tous les outils nécessaires et créer votre premier repo GitHub.

## Livrables

1. VSCode installé avec extensions recommandées
2. Git configuré (username, email)
3. GitHub account créé
4. Repository "DevSetup" créé avec
- Fichier README.md bien documenté
- Dossier "notes" avec 3 fichiers markdown
- Fichier .gitignore standard
5. Premier commit poussé sur GitHub


## Critères d'évaluation


✓ VSCode fonctionne correctement
✓ Commandes Git (init, add, commit, push) maîtrisées
✓ Repository visible et public sur GitHub
✓ Historique Git propre (3+ commits)
✓ README explicite comment utiliser le repo

## BONUS
✓ Configurer Nodemon pour développement
✓ Ajouter un .editorconfig au projet
✓ Créer une branche "dev" et merger dans main
